Cara Test Tanpa Cloudflare Cache:

Option 1: Edit hosts file (RECOMMENDED)

# Mac/Linux - Edit /etc/hosts
sudo nano /etc/hosts

# Tambahkan baris ini:
159.65.134.137 www.aurapass.xyz

# Save (Ctrl+O, Enter, Ctrl+X)
# Flush DNS:
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder

# Sekarang buka: http://www.aurapass.xyz
# (akan langsung ke VPS, bypass Cloudflare)

Option 2: Purge Cloudflare Cache

  1. Login: https://dash.cloudflare.com
  2. Pilih domain: aurapass.xyz
  3. Klik: Caching → Configuration
  4. Klik: "Purge Everything"
  5. Tunggu 30 detik
  6. Hard refresh browser: Cmd+Shift+R

Option 3: Development Mode (Temporary)

  1. Cloudflare dashboard → Caching
  2. Enable "Development Mode" (3 jam bypass cache)
  3. Refresh browser
